# Chocolate Shipment Data Analysis

Exploratory analysis of chocolate shipment logs to identify delay causes, seasonality, route-level KPIs, and recommendations to improve on-time delivery.

Repository highlights
- Notebooks: `notebooks/` (EDA.ipynb, Aggregation.ipynb, Recommendations.ipynb)  
- Reports: `reports/` (dashboard PBIX, screenshots, executive summary)  
- Data: `data/` (provide small sample CSVs or schema)

Tech
-  Power BI

Quick summary
- Goal: Surface operational bottlenecks in shipment data and propose actionable routing/scheduling improvements.  
- Key outputs: delay distribution by route, seasonal trends, top causes of delay, and dashboards for ops teams.

How to view the Power BI dashboard locally
1. Install Power BI Desktop: https://powerbi.microsoft.com/  
2. Open `reports/chocolate_shipment_dashboard.pbix`.  
3. If the PBIX expects CSV data, place the sample CSV at `data/chocolate_shipments_sample.csv` or update the data source in Power Query.  
4. Refresh visuals and explore the Report pages (Route KPIs, Delay Heatmap, Trend analysis).
